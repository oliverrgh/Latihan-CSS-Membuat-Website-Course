Pengenalan CSS

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mengatur tampilan dan desain halaman web. Dengan CSS, kita bisa mengontrol warna, font, ukuran, tata letak, hingga efek animasi pada elemen HTML.

Struktur Dasar CSS

            selector {
                property: value;
            }

Selector digunakan untuk memilih elemen HTML yang ingin diubah, property adalah atribut tampilan yang diatur, dan value adalah nilai yang diberikan.

Cara Menghubungkan CSS

  • Inline CSS: ditulis langsung pada atribut style di elemen HTML.
  • Internal CSS: ditulis di dalam tag <style> di bagian <head>.
  • External CSS: ditulis di file terpisah (.css) lalu dihubungkan dengan <link>.

Contoh

            /* External CSS (style.css) */
            body {
                background-color: #f0f0f0;
                font-family: Arial, sans-serif;
            }

            h1 {
                color: blue;
                text-align: center;
            }

            p {
                font-size: 16px;
                line-height: 1.5;
            }

Selektor Umum

  • Elemen: memilih berdasarkan tag (contoh: p, h1).
  • Kelas (.class): memilih berdasarkan atribut class.
  • ID (#id): memilih berdasarkan atribut id.
  • Universal (*): memilih semua elemen.

Kesimpulan

CSS sangat penting untuk mempercantik dan memperjelas tampilan website. Setelah memahami CSS dasar, kita bisa lanjut ke topik lanjutan seperti Flexbox, Grid, dan Animasi.
