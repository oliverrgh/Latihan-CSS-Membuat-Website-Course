Pengenalan JavaScript

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat halaman web menjadi interaktif dan dinamis. Dengan JavaScript, kita bisa menambahkan logika, animasi, validasi form, hingga komunikasi dengan server.

Contoh Dasar JavaScript

            <script>
                // Menampilkan pesan di console
                console.log("Halo dunia!");

                // Membuat interaksi dengan user
                alert("Selamat datang di website ini!");
            </script>

Fungsi JavaScript

            function sapa(nama) {
                return "Halo, " + nama + "!";
            }

            console.log(sapa("Oliver")); // Output: Halo, Oliver!

DOM (Document Object Model)

JavaScript bisa mengakses dan mengubah elemen HTML melalui DOM. 

            <button onclick="ubahTeks()">Klik Saya</button>

            <p id="pesan">Teks awal</p>

            <script>
                function ubahTeks() {
                    document.getElementById("pesan").innerText = "Teks sudah berubah!";
                }
            </script>

Kesimpulan

JavaScript adalah bahasa pemrograman inti di web modern. Setelah menguasai dasar-dasarnya, kamu bisa melanjutkan ke topik seperti ES6+, API, Asynchronous JavaScript (Promise & Async/Await), serta framework populer seperti React, Vue, atau Angular.
