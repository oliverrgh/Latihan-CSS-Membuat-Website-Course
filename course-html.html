Pengenalan HTML

HTML (HyperText Markup Language) adalah bahasa standar untuk membuat struktur halaman web. Dengan HTML, kita bisa menentukan elemen-elemen penting seperti heading, paragraf, link, gambar, hingga tabel dan form.

Struktur Dasar HTML

            <!DOCTYPE html>
            <html lang="en">
            <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Judul Halaman</title>
            </head>
            <body>
            <h1>Halo Dunia!</h1>
            <p>Ini adalah paragraf pertama saya.</p>
            </body>
            </html>

Elemen Penting

  • <h1> - <h6>: Heading atau judul.
  • <p>: Paragraf teks.
  • <a>: Hyperlink.
  • <img>: Gambar.
  • <ul>, <ol>, <li>: Daftar.

Contoh

            <h1>Belajar HTML</h1>
            <p>HTML itu mudah dan menyenangkan!</p>
            <a href="https://www.w3schools.com">Belajar lebih lanjut</a>
            <img src="https://via.placeholder.com/150" alt="Contoh gambar">

Kesimpulan

HTML adalah pondasi utama dalam pembuatan website. Setelah memahami HTML, lanjutkan ke CSS untuk mempercantik tampilan, dan JavaScript untuk membuat halaman interaktif.
